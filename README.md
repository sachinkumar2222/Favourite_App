## üì• Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/sachinkumar2222/Favourite_App.git
    ```
2.  **Open in Android Studio**:
    Open the project directory.
3.  **Add API Key**:
    Get your API Key from [TMDB](https://www.themoviedb.org/settings/api) and add it to your project configuration (or `MovieViewModel` if hardcoded for demo).
4.  **Build & Run**:
    Connect your device/emulator and hit Run.
